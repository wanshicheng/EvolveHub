## 9. 初始化步骤

### Step 1: 后端项目骨架
1. 创建 `backend/pom.xml` 父 POM
2. 创建 `evolvehub-common` 公共模块
3. 创建 `evolvehub-gateway` 网关模块
4. 创建 `evolvehub-auth` 认证模块
5. 创建 `evolvehub-ai` AI 中台模块
6. 创建 `evolvehub-admin` 管理模块
7. 各模块 `application.yml` 配置

### Step 2: 前端项目骨架
1. Vite + Vue 3 + TypeScript 初始化
2. 安装 Element Plus / Pinia / Vue Router / Axios
3. 配置路由、状态管理、API 封装
4. 创建基础布局和登录页

### Step 3: Docker 部署文件
1. `docker-compose.yml` 编排文件
2. MySQL 初始化脚本
3. Redis 配置
4. Nginx 反向代理配置

### Step 4: 验证
1. 后端各服务能独立启动
2. Gateway 能正确路由到各服务
3. 前端能连接后端 API
4. Docker Compose 一键启动
